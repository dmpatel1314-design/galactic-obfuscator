galactic-obfuscator.github.io

Project page for the obfuscator prototype

About

This is a minimal prototype demonstrating:

  • JavaScript obfuscation (AST-based) — identifier renaming + string literal encoding
  • Lua obfuscation (packer) — comment stripping + XOR + base64 loader

Quick start

Clone, install and run the CLI:

git clone git@github.com:dmpatel1314-design/galactic-obfuscator.git
cd galactic-obfuscator
npm install
node ./src/cli.js example.js -o example.obf.js

For Lua:

node ./src/cli.js example.lua --lang lua -o example.obf.lua
